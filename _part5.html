Glass Effect Recipe

Superficie Glass
Este panel usa backdrop-filter:blur(12px) sobre el gradiente navy del fondo. Nota cómo el fondo se desenfoca a través del panel.
.glass-surface {
  background: rgba(232, 236, 244, 0.72);     /* --glass-bg      */
  backdrop-filter: blur(12px);
  -webkit-backdrop-filter: blur(12px);
  border: 1px solid rgba(255, 255, 255, 0.60); /* --glass-border  */
  box-shadow:
    inset 0 1px 0 rgba(255,255,255,0.95),       /* --glass-shine   */
    var(--neu-out-sm);                           /* elevación suave */
  border-radius: 16px;
}

/* IMPORTANTE: requiere fondo con variación (gradiente o imagen)
   No visible sobre fondo plano --bg #e8ecf4 */
Cuándo usar Glass
Modales sobre contenido con gradiente · Overlays sobre dashboards con gráficas · Panels flotantes en pantallas con background imagen. No usar sobre fondo plano --bg — el blur no es visible.

Secuencia obligatoria para todos los elementos interactivos
Default → Hover → Active → Focus → Disabled. Nunca omitas el estado Focus (accesibilidad). Disabled: agrega cursor:not-allowed y pointer-events:none.

Color Usage Guidelines

SÍ hacer
  • --navy para títulos principales y valores importantes
  • --indigo para elementos interactivos y brand
  • --text-sub para texto secundario y labels
  • --text-dim para metadatos y timestamps
  • Semánticos (green/yellow/red) solo para estados
NO hacer
  • Mezclar más de 2 colores brand en un componente
  • --red para estados no-error
  • --orange como color de acción (solo acento visual)
  • Texto --text-dim en tamaño menor a 11px
  • Usar --navy directo sobre --bg en texto pequeño (bajo contraste)
